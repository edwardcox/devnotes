## ğŸ“– Usage

### Getting Started
1. Open the file in any modern browser
2. Click "+ New Note" or press `Ctrl/Cmd + N`
3. Select a category and add tags
4. Write your content with markdown or code
5. Toggle preview mode with `Ctrl/Cmd + E`
