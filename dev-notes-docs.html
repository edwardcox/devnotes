What is Dev Notes?

Dev Notes is a powerful, portable, and privacy-focused note-taking application designed specifically for developers. It's a single HTML file that contains a complete note management system with syntax highlighting, markdown support, custom categories, and persistent storage.

🎯 Key Characteristics
  • Single File: The entire application is contained in one HTML file (~2,500 lines)
  • No Dependencies: No npm, no build process, no external libraries needed
  • Works Offline: Full functionality without internet connection
  • Privacy First: All data stored locally in browser storage
  • Portable: Email it, USB it, share it - works everywhere

Why Dev Notes?

Dev Notes was created to solve common problems developers face with note-taking:

🔒
Privacy & Security

📦

Who is it for?

Dev Notes is perfect for:

  • Developers who need to store code snippets, API keys, and configurations
  • System Administrators managing server configs and commands
  • DevOps Engineers documenting deployment procedures
  • Technical Writers creating documentation with code examples
  • Students learning programming and taking technical notes

🚀 Quick Start Guide

Get up and running with Dev Notes in under 60 seconds!

Step 1: Download

Download the dev-notes-v3.5.html file to your computer.

Step 2: Open

Double-click the file to open it in your web browser (Chrome, Firefox, Edge, Safari all work).

Step 3: Create Your First Note

  1. Click + New Note or press Ctrl + N
  2. Give your note a title
  3. Select a category (API, Config, Snippet, etc.)
  4. Choose a language for syntax highlighting
  5. Start typing!
💡 Quick Tips
  • Press ? to see all keyboard shortcuts
  • Press Ctrl + D to toggle dark/light theme
  • Press Ctrl + E to preview markdown
  • Your notes are automatically saved to browser storage

Storage Implementation

localStorage Structure

Dev Notes uses localStorage with the following keys:

// Notes storage
localStorage.setItem('devNotes_v3', JSON.stringify([
    {
        id: 1234567890,           // timestamp-based ID
        title: "Note Title",
        content: "Note content",
        category: "api",
        tags: ["tag1", "tag2"],
        language: "javascript",
        createdAt: "2024-01-01T00:00:00.000Z",
        updatedAt: "2024-01-01T00:00:00.000Z"
    }
]));

// Categories storage
localStorage.setItem('devNotes_categories_v1', JSON.stringify([
    {
        id: "api",
        label: "API",
        icon: "🔌",
        color: "#f59e0b"
    }
]));

// Theme storage
localStorage.setItem('devNotes_theme_v1', 'dark'); // or 'light'

Storage Manager Class

class StorageManager {
    constructor() {
        this.notesKey = 'devNotes_v3';
        this.categoriesKey = 'devNotes_categories_v1';
        this.themeKey = 'devNotes_theme_v1';
        this.isStorageAvailable = this.checkStorageAvailability();
    }
    
    save(notes) {
        if (this.isStorageAvailable) {
            localStorage.setItem(this.notesKey, JSON.stringify(notes));
            return true;
        }
        // Fallback to in-memory storage
        window.__tempNotes = notes;
        return true;
    }
    
    load() {
        if (this.isStorageAvailable) {
            const data = localStorage.getItem(this.notesKey);
            return data ? JSON.parse(data) : null;
        }
        return window.__tempNotes || null;
    }
}

Storage Limits

  • localStorage typically allows 5-10MB per origin
  • Each note is approximately 2-5KB
  • Can store approximately 1,000-2,000 notes
  • Includes storage size indicator in UI

❓ Frequently Asked Questions

Q: Where is my data stored?

Your data is stored locally in your browser's localStorage. It never leaves your device and is not transmitted to any server.

Q: What happens if I clear my browser data?

Clearing browser data will delete your notes. Always use the Export feature to create backups of important notes.

Q: Can I use Dev Notes on multiple devices?

Yes! Use the Export feature on one device and Import on another. You can sync via any file sharing service (Dropbox, Google Drive, etc.).

Q: How many notes can I store?

Typically 1,000-2,000 notes, depending on size. localStorage usually provides 5-10MB of storage per domain.

Q: Can I collaborate with others?

Dev Notes is designed for personal use. For collaboration, you can export and share JSON files.

Q: Does it work offline?

Yes! Dev Notes works completely offline. No internet connection is required.

Q: Can I modify the source code?

Absolutely! Dev Notes is provided as-is for you to use and modify as needed.

Q: Is there a mobile app?

Dev Notes works in mobile browsers. You can add it to your home screen for an app-like experience.

Q: How do I report bugs or suggest features?

Since Dev Notes is a standalone application, you can modify it directly to suit your needs.

Q: What's the difference between categories and tags?

Categories are predefined groups (like API, Config, etc.) while tags are flexible labels you create on-the-fly for any purpose.

Troubleshooting

Common Issues and Solutions:
Issue Solution
Notes not saving Check if localStorage is enabled in your browser settings
Can't open file directly Some browsers block local file access. Use a local server or different browser
Theme not persisting Ensure cookies and site data are not being cleared on browser close
Keyboard shortcuts not working Ensure no browser extensions are intercepting the shortcuts
Import failing Verify the JSON file is valid and was exported from Dev Notes
